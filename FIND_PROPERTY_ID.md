# How to Find Your GA4 Property ID

Your Measurement ID is: **G-Z7697T1078**

To find your Property ID (the number we need):

## Method 1: From Google Analytics Dashboard (Easiest)

1. Go to [Google Analytics](https://analytics.google.com/)
2. Click **Admin** (gear icon, bottom left)
3. In the **Property** column, click **"Property Settings"**
4. Look for **"Property ID"** - it's a number like `123456789` (usually 8-9 digits)
5. Copy that number

## Method 2: From URL

1. When you're in Google Analytics, look at the URL
2. It might look like: `https://analytics.google.com/analytics/web/#/p123456789/...`
3. The number after `/p` is your Property ID
